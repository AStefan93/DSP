DSP

Digital Signal Processing Overall Project

#Dependancies: OpenCV 3.0+

Install VMware Ubuntu from SSP7. Request admin rights for VM.

Make sure you have internet access on the VM. If you don't, call service IT (1300).

Run in terminal: sudo apt-get update

sudo apt-get upgrade

sudo apt-get install git

sudo apt-get install codeblocks

This will install the prebuilt OpenCV library for Ubuntu (it is usually relatively old). It will be sufficient for now.

sudo apt-get install libopencv-dev python-opencv

This step will be updated in time to install the latest OpenCV library.

Create a "Work" directory on your VM. (mkdir Work) Go to this folder (cd Work) and run:

git clone –recurse-submodules http://github.conti.de/uidg6179/DSP.git

If you used "git clone", just use the command "git submodule update --init" afterwards.

After downloading all the files, open CodeBlocks and select "Open existing project". Select the project located in the "DSP/src" folder.

If everything went well, you should be able to build and run the code on your VM. Happy coding.

In order to make sure the repository is up to date with the main branch:

This will show you the differences between local repository and branch:

git status

This will update your local repository with the commited versions from branch:

git pull –recurse-submodules

In order to commit something in your submodule:

Create a local repository for your own submodule and after modifying the files:

(Optional)**git status**

git add *

git commit -m "insert message"

git push

To update submodule versions in higher project:

cd submodule-name

git checkout <tagname>

cd ..

git add submodule-name

git commit -m "updating submodule to <tagname>"
